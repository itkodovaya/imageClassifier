Обучение нейросети на военной технике
=====================================

Структура папок:
- bpla/        БПЛА, дроны
- aircrafts/  Самолёты
- birds/       Птицы (для отличия от БПЛА)
- tanks/       Танки
- helicopters/ Вертолёты
- ships/       Корабли
- apc/         БТР, БМП
- trucks/      Военные грузовики
- artillery/   Артиллерия, САУ
- submarines/  Подводные лодки

Инструкция:
1. Добавьте изображения (.jpg, .png) в каждую папку
2. Минимум 30-50 фото на класс, желательно 100+
3. Запустите ImageClassifierWithTraining.exe
4. Training -> Select Folder -> выберите эту папку
5. Start Training
